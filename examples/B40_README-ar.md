# الحزمة B-040: قاموس ar-v1.1 — متزامن وطابق وحالة

## نظرة عامة

تضيف هذه الحزمة البنية التحتية لإصدارات القواميس وتُصدر القاموس `ar-v1.1`،
وهو مجموعة عليا صارمة من `ar-v1` تضيف كلمة مفتاحية واحدة جديدة:

| Python | ar-v1 | ar-v1.1 الجديد |
|---|---|---|
| `async` | `غير_متزامن` | `متزامن` ✨ (أيضاً) |

الكلمات `انتظر` و`طابق` و`حالة` موجودة أصلاً في `ar-v1`؛ يوثّق هذا الملف
استخدامها معاً.

---

## الكلمة الجديدة: متزامن

`متزامن` تعني "متزامن / متوازٍ" بالعربية الفصحى المعيارية، وهي مصطلح أكاديمي
راسخ في أدبيات علوم الحاسوب العربية. تُعدّ بديلاً أقصر وأوضح من `غير_متزامن`
التي تعني حرفياً "غير متزامن" وتبدو نفياً.

كلتا الصيغتين مقبولتان في `ar-v1.1`:

```python
# ar-v1 — لا تزال تعمل
غير_متزامن دالة f():
    انتظر g()

# ar-v1.1 — الصيغة الأقصر الموصى بها
متزامن دالة f():
    انتظر g()
```

## مثال 1: البرمجة غير المتزامنة (`B40_async_demo.apy`)

يوضح هذا المثال:
- تعريف دوال متزامنة بـ `متزامن دالة`
- انتظار النتائج بـ `انتظر`
- تشغيل مهام متعددة بـ `asyncio.gather`
- التكرار غير المتزامن بـ `متزامن لكل`

```python
# arabicpython: dict=ar-v1.1
استورد asyncio

متزامن دالة جلب_بيانات(معرف):
    انتظر asyncio.sleep(0)
    ارجع معرف * 10

متزامن دالة رئيسيه():
    نتائج = انتظر asyncio.gather(
        جلب_بيانات(1),
        جلب_بيانات(2),
        جلب_بيانات(3),
    )
    اطبع("النتائج:", نتائج)

asyncio.run(رئيسيه())
```

تشغيل المثال:

```bash
ثعبان --dict ar-v1.1 examples/B40_async_demo.apy
# أو لأن الملف يحتوي على التوجيه:
ثعبان examples/B40_async_demo.apy
```

## مثال 2: مطابقة الأنماط (`B40_match_demo.apy`)

تُعدّ `طابق` و`حالة` كلمتين مفتاحيتين لينتين تتيحان مطابقة الأنماط الهيكلية
(PEP 634، Python 3.10+):

```python
دالة صف_الشكل(شكل):
    طابق شكل:
        حالة {"نوع": "دائره", "نصف_قطر": ق}:
            ارجع f"دائرة نصف قطرها {ق}"
        حالة {"نوع": "مستطيل", "عرض": ع, "ارتفاع": ا}:
            ارجع f"مستطيل {ع}×{ا}"
        حالة _:
            ارجع "شكل غير معروف"
```

**ملاحظة**: بما أن `طابق` كلمة مفتاحية لينة في Python، يمكن استخدامها اسماً
للمتغير دون خطأ:

```python
طابق = 5   # صحيح تماماً
```

## التوافق مع الإصدارات السابقة

برامج `ar-v1` **لا تتأثر** بهذه الحزمة:

- ثعبان بدون `--dict` يستخدم `ar-v2` (السلوك الافتراضي الحالي)
- `غير_متزامن` لا تزال تعمل في `ar-v1` و`ar-v1.1` و`ar-v2`
- لا يوجد تعارض: `متزامن` غير موجودة في `ar-v1` أو `ar-v2`

## مرجع سريع

| Python | العربية | الإصدار |
|---|---|---|
| `async def` | `متزامن دالة` | ar-v1.1 |
| `async def` | `غير_متزامن دالة` | ar-v1 ، ar-v1.1 ، ar-v2 |
| `await` | `انتظر` | ar-v1 ، ar-v1.1 ، ar-v2 |
| `async for` | `متزامن لكل` | ar-v1.1 |
| `async with` | `متزامن مع` | ar-v1.1 |
| `match x:` | `طابق x:` | ar-v1 ، ar-v1.1 ، ar-v2 |
| `case 1:` | `حالة 1:` | ar-v1 ، ar-v1.1 ، ar-v2 |
