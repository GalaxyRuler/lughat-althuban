# التطبيع وسلامة اتجاه النص

يطبّع وقت التشغيل المعرّفات العربية قبل البحث في القاموس. الهدف هو قبول الكتابة الطبيعية دون فتح باب لتصادمات خفية.

## ما يطبّعه النظام

- الهمزات: `أ` و`إ` و`آ` إلى `ا`.
- التاء المربوطة النهائية إلى هاء.
- الألف المقصورة إلى ياء.
- الحركات والتطويل.

## ما يرفضه النظام

- محارف التحكم باتجاه النص خارج النصوص الحرفية والتعليقات.
- مزج أكثر من نظام أرقام داخل العدد الواحد، مثل `١2`.

## أمثلة

```python
اطبع(خطأ)
اطبع(خطا)      # يطبع الاسم نفسه بعد التطبيع إذا كان في موضع معرّف
```

```python
عدد = ١٢٣      # صالح
عدد = ١2       # مرفوض
```
