# سياسة التسمية العربية

يعتمد المشروع معجما واحدا في `lexicon/`. أي مصطلح جديد يبدأ هناك ثم ينتقل إلى القواميس والوثائق والأمثلة عبر التوليد أو التحقق.

## القواعد

- استخدم العربية الفصحى الواضحة حيث أمكن.
- استخدم التعريب الصوتي فقط عندما يكون المصطلح اسما راسخا مثل `لامدا` أو اسم مكتبة مثل `نمباي`.
- استخدم الشرطة السفلية في العبارات المركبة: `خطأ_قسمة_صفر`.
- اكتب الشكل المرئي الطبيعي في المعجم، حتى لو كان وقت التشغيل يطبّع الهمزات أو التاء المربوطة.
- لا تضف مرادفا جديدا في وقت التشغيل إلا بقرار توافق، لأن كل اسم جديد يصبح جزءا من واجهة المستخدم.

## القبول

لا يقبل مصطلح جديد حتى تمر الأوامر:

```bash
python tools/validate_lexicon.py
python tools/generate_lexicon_outputs.py --check
python -m pytest
```
