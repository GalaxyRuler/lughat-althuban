# الحدود المعروفة

- لا يترجم المشروع رسائل CPython كلها؛ الرسائل الشائعة فقط لها قوالب عربية.
- بعض مكتبات الطرف الثالث تحتاج اعتمادا اختياريا قبل تشغيل أمثلتها.
- ليست كل طرق الكائنات داخل مكتبات البيانات والويب معرّبة بعد.
- دعم RTL في المحررات يعتمد على VS Code أو Jupyter نفسه، وليس على لغة الثعبان وحدها.
- `pip show` في طرفيات Windows غير UTF-8 قد يفشل في طباعة بيانات الحزمة العربية.
- أسماء Python التقنية داخل الشفرة المدمجة أو المكتبات تبقى كما هي حفاظا على التوافق.
