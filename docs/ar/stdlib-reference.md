# مرجع المكتبة القياسية

هذا المرجع موجز للوحدات القياسية المتاحة بأسماء عربية. مصدر الحقيقة هو `lexicon/libraries.toml`، والملفات الناتجة في `arabicpython/aliases/*.toml` و`docs/ar/aliases-index.md` تولد منه.

## حزمة بيئة النظام

| الاسم العربي | Python | مثال |
|--------------|--------|------|
| `نظام` | `os` | `استورد نظام` |
| `مسار` | `pathlib` | `استورد مسار` |
| `مسار_نظام` | `os.path` | `استورد مسار_نظام` |
| `نظام_بايثون` | `sys` | `استورد نظام_بايثون` |

```python
استورد نظام
استورد مسار

اطبع(نظام.الدليل_الحالي())
ملف = مسار.مسار("README.md")
اطبع(ملف.موجود())
```

## حزمة البيانات والنصوص

| الاسم العربي | Python | مثال |
|--------------|--------|------|
| `جيسون` | `json` | `جيسون.نص({...})` |
| `ملف_csv` | `csv` | قراءة وكتابة CSV |
| `قاعدة_بيانات` | `sqlite3` | SQLite المدمجة |
| `تعابير` | `re` | `تعابير.بحث(...)` |
| `نصوص` | `string` | ثوابت وقوالب نصية |
| `تنسيق_نص` | `textwrap` | تنسيق الفقرات |

```python
استورد جيسون
استورد تعابير

نص = جيسون.نص({"اسم": "لغة الثعبان"}, ensure_ascii=خطأ)
اطبع(نص)
اطبع(تعابير.بحث(r"\w+", نص).مجموعه())
```
