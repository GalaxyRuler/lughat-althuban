# دليل البدء

أهلاً بك في دليل البدء السريع للغة الثعبان. سيأخذك هذا الدليل في جولة تدريجية عبر 10 أقسام لتتعلم كيفية كتابة برامجك الأولى بالعربية بالكامل.

## 1. التثبيت والتحقق

تأكد من توافر بايثون (Python) لديك، ثم قم بتثبيت لغة الثعبان. افتح سطر الأوامر ونفذ:
```bash
git clone https://github.com/GalaxyRuler/lughat-althuban.git
cd lughat-althuban
pip install -e .
```
للتأكد من نجاح التثبيت، جرب تشغيل الواجهة التفاعلية بكتابة الأمر `ثعبان` في سطر الأوامر.

## 2. مرحبا بالعالم

اكتب أول سطر لك بالثعبان. قم بإنشاء ملف باسم `بداية.apy` واكتب فيه:
```python
اطبع("مرحبا بالعالم")
```
لتشغيله، نفذ الأمر: `ثعبان بداية.apy`.

## 3. المتغيرات والحساب

استخدام المتغيرات والعمليات الحسابية يتم بسلاسة:
```python
سعر = 100
ضريبة = 15
سعر_نهائي = سعر + (سعر * ضريبة / 100)
اطبع(سعر_نهائي)
```
*ملاحظة:* لغة الثعبان تدعم نفس أنواع البيانات القياسية كالأعداد الصحيحة (صحيح) والنصوص (نص).

## 3.5. اختصارات نحوية (بادئات النصوص والأرقام، وتسلسلات الهروب)

### بادئات النصوص

يمكنك استخدام أحرف عربية كبادئات للنصوص بدلاً من الأحرف الإنجليزية:

- `ت"..."` → `f"..."` (نص معرف: تنسيق)
- `ب"..."` → `b"..."` (بايتات)
- `خ"..."` → `r"..."` (خام)
- `ي"..."` → `u"..."` (يونيكود)

يمكنك أيضاً الدمج: `خت"مسار\سلاحقة.txt"` تعادل `rf"مسار\nلاحقة.txt"`.

### تسلسلات الهروب العربية

في النصوص، استخدم أحرفاً عربية بدلاً من التسلسلات الإنجليزية:

- `\س` → `\n` (سطر جديد)
- `\ج` → `\t` (تبويب)
- `\ر` → `\r` (العودة للبداية)
- `\م` → `\b` (محو للخلف)
- `\ف` → `\f` (فاصل صفحة)
- `\ع` → `\v` (تبويب عمودي)
- `\ن` → `\a` (رنين)

مثال: `"مرحبا\sالعالم"` ينتج سطرين بدلاً من سطر واحد.

### بادئات الأرقام

استخدم أحرفاً عربية لكتابة الأرقام الدقيقة والثنائية والثمانية:

- `0سFF` → `0xFF` (ستة عشر / سادس عشر)
- `0ث1010` → `0b1010` (ثنائي)
- `0ذ17` → `0o17` (ذو ثمانية)

مثال: `لون = 0سABCDEF` لون مثل `lون = 0xABCDEF`.

## 4. مسار التحكم (اذا/لكل/طالما)

استخدم الشروط وحلقات التكرار للتحكم في مسار البرنامج:
```python
لكل رقم في نطاق(5):
    اذا رقم > 2:
        اطبع("رقم كبير")
    والا:
        اطبع("رقم صغير")
```

## 5. الدوال والأصناف

يمكنك تغليف شفرتك باستخدام الدوال والأصناف بسهولة:
```python
صنف سيارة:
    دالة تهيئة(نفسي، نوع):
        نفسي.نوع = نوع
    
    دالة اعرض(نفسي):
        اطبع(نفسي.نوع)

سيارتي = سيارة("رياضية")
سيارتي.اعرض()
```

## 6. التعامل مع الملفات

تسهل وحدة `مسار` (pathlib) التعامل مع الملفات:
```python
استورد مسار

ملف = مسار.مسار("تجربة.txt")
ملف.اكتب_نص("هذا ملف تجريبي")
اطبع(ملف.اقرا_نص())
```

## 7. قواعد البيانات

يمكنك إنشاء وإدارة قواعد البيانات المدمجة باستخدام وحدة `قاعدة_بيانات` (sqlite3):
```python
استورد قاعدة_بيانات

اتصال = قاعدة_بيانات.اتصل(":memory:")
اتصال.نفذ("CREATE TABLE طلاب (اسم TEXT)")
اتصال.نفذ("INSERT INTO طلاب VALUES ('أحمد')")
لكل صف في اتصال.نفذ("SELECT * FROM طلاب"):
    اطبع(صف)
```

## 8. البرمجة غير المتزامنة

لبرمجة تطبيقات عالية الأداء، تدعم اللغة مفاهيم التزامن عبر وحدة `اتزامن` (asyncio):
```python
استورد اتزامن

غير_متزامن دالة رئيسية():
    نتائج = انتظر اتزامن.اجمع_مهام(اتزامن.تريث(1), اتزامن.تريث(2))
    اطبع("انتهت المهام المتزامنة")

اتزامن.شغل(رئيسية())
```

## 9. قراءة رسائل الخطأ

عند حدوث خطأ، توفر لك لغة الثعبان رسائل خطأ معربة بالكامل لتسهيل فهم المشكلة:
```python
حاول:
    قائمتي = [1, 2]
    اطبع(قائمتي[5])
استثناء خطأ_فهرس باسم خطأ:
    اطبع("عذراً، الفهرس غير موجود!")
```
