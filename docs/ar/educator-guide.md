# دليل المعلم

ابدأ بالأمثلة السبعة الأولى في `examples/`، ثم انتقل إلى المكتبة القياسية، ثم إلى أمثلة البيانات والويب.

## مسار مقترح

1. `01_hello.apy`: الطباعة والنصوص.
2. `03_control_flow.apy`: الشروط والتكرار.
3. `04_functions.apy`: الدوال والقيم الراجعة.
4. `06_classes.apy`: الأصناف.
5. `07_imports.apy`: تقسيم البرنامج إلى ملفات.
6. `B37_async_demo.apy`: البرمجة غير المتزامنة.
7. `B40_match_demo.apy`: المطابقة الهيكلية.

## نصائح

- اعرض مكافئ Python عند الحاجة، لكن اجعل الشفرة الأولى عربية.
- استخدم [glossary.md](glossary.md) لتوحيد المصطلحات داخل الصف.
- اشرح التطبيع مبكرا حتى يفهم الطلاب لماذا تعمل بعض فروق الهمزات.
