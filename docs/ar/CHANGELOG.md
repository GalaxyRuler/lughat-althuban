## المرحلة أ (مكتملة)
شهدت هذه المرحلة إرساء الأساس القوي للغة وتوفير البنية التحتية الرئيسية:
- **الكلمات المفتاحية**: ترجمة كافة الكلمات المفتاحية الأساسية مثل (`دالة`, `صنف`, `اذا`, `لكل`, `طالما`, `ارجع`, `ارفع`, `حاول`, `اخيرا`, `مع`, `انتظر`, `غير_متزامن`, `سلم`, `مقاطعه`).
- **الدوال المدمجة**: توفير مرادفات عربية أصيلة مثل (`اطبع`, `نطاق`, `قائمه`, `قاموس`, `نص`, `صحيح`, `طول`).
- **الاستثناءات**: ترجمة شاملة لأخطاء النظام الشائعة (`خطأ_قيمة`, `خطأ_نوع`, `خطأ_فهرس`, `خطأ_مفتاح`).
- **واجهات التشغيل**: بناء آليات تنفيذ مرنة تشمل تشغيل الملفات `ثعبان ملف.apy`، الأوامر النصية `ثعبان -c`، والواجهة التفاعلية المباشرة (REPL).
- **رسائل الخطأ**: تعريب أخطاء التشغيل لتسهيل تتبع المشاكل وفهمها للمبرمج العربي.
