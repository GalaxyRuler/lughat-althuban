```python
موظفون = بد.اطار_بيانات({
    "الاسم": ["أحمد", "سارة", "محمد", "فاطمة"],
    "القسم": ["تقنية", "تسويق", "تقنية", "موارد_بشرية"],

    "سنوات_الخبرة": [3, 5, 7, 2]
})

# التصفية

اطبع("قسم التقنية:\n", تقنية)

# التجميع
متوسط_بالقسم = موظفون.groupby("القسم")["الراتب"].mean()
اطبع("\nمتوسط الراتب بالقسم:\n", متوسط_بالقسم)

# الإحصاء الوصفي
اطبع("\nالإحصاء الوصفي:\n", موظفون["الراتب"].describe())
```

## سايباي (`scipy`)

### الاستيراد

```python
استورد سايباي باسم عح
من سايباي استورد سايباي.stats باسم احصائيات_علميه
```

### الوحدات الفرعية

| Python | العربية | الوصف |
|--------|---------|--------|
| `linalg` | `جبر_خطي_علمي` | الجبر الخطي المتقدم |
| `special` | `دالات_خاصه` | الدوال الرياضية الخاصة |
| `optimize` | `تحسين` | تحسين الدوال وإيجاد الجذور |
| `integrate` | `تكامل` | التكامل العددي |
| `signal` | `اشارات` | معالجة الإشارات |
| `stats` | `احصائيات_علميه` | الاختبارات الإحصائية |
| `interpolate` | `استيفاء` | الاستيفاء العددي |
| `fft` | `تحويل_فورييه` | تحويل فورييه السريع |
| `sparse` | `مصفوفات_نادره` | المصفوفات المتفرقة |

### مثال

```python
من scipy import optimize as محسّن
من scipy import stats as احص

# إيجاد جذر دالة
دالة مربع_ناقص_4(س):
    ارجع س**2 - 4

جذر = محسّن.brentq(مربع_ناقص_4, 0, 3)
اطبع(f"جذر المعادلة س²-4=0: {جذر}")  # 2.0

# اختبار t
مجموعه_1 = [23, 25, 28, 22, 27]
مجموعه_2 = [30, 32, 28, 35, 31]
t_stat, p_value = احص.ttest_ind(مجموعه_1, مجموعه_2)
اطبع(f"قيمة t: {t_stat:.3f}, قيمة p: {p_value:.3f}")
```

## صور (`PIL.Image`)

### الاستيراد

```python
استورد صور
```

### الوظائف الرئيسية

| Python | العربية | الوصف |
|--------|---------|--------|
| `open` | `افتح_صوره` | فتح ملف صورة |
| `new` | `صوره_جديده` | إنشاء صورة جديدة |

### مثال

```python
استورد صور

# فتح صورة وتحجيمها

اطبع(f"الوضع: {صوره.mode}")

# تحجيم
صوره_مصغره = صوره.resize((200, 200), صور.تحجيم_عالي_جوده)
صوره_مصغره.save("مصغره.jpg")

# تحويل لتدرج رمادي
رمادي = صوره.convert("L")
رمادي.save("رمادي.jpg")
```

# Science & Data Aliases (English summary)

| Arabic import | Python module | Purpose |
|---|---|---|
| `نمباي` | `numpy` | N-dimensional arrays, linear algebra, FFT |
| `بانداس` | `pandas` | DataFrames, CSV/Excel I/O, groupby |
| `سيبورن` | `seaborn` | Statistical visualisation on top of matplotlib |
| `سايباي` | `scipy` | Optimize, integrate, signal, stats |
| `صور` | `PIL.Image` | Image open/save/resize (Pillow) |

All imports use `استورد <arabic_name>`. Aliases are resolved by the alias runtime (B-001). These packages must be installed separately — they are not bundled with لغة الثعبان.
