# مكتبات الويب — Web Aliases

لغة الثعبان توفر أسماء عربية لأبرز مكتبات تطوير الويب وطلبات HTTP.

---

## جدول المكتبات

| الاسم العربي | Python | الوصف |
|-------------|--------|--------|
| `فلاسك` | `flask` | إطار الويب الخفيف |
| `طلبات` | `requests` | طلبات HTTP المتزامنة |
| `أيو_هتب` | `aiohttp` | طلبات HTTP غير المتزامنة |
| `فاست_أبي` | `fastapi` | إطار API سريع مبني على Pydantic |

## فلاسك (`flask`)

### الاستيراد

```python
استورد فلاسك
```

### المكونات الرئيسية

| Python | العربية | الوصف |
|--------|---------|--------|
| `Flask` | `فلاسك.فلاسك` | الصنف الرئيسي للتطبيق |
| `Blueprint` | `فلاسك.مخطط` | تنظيم المسارات |
| `request` | `فلاسك.طلب` | كائن الطلب الحالي |
| `session` | `فلاسك.جلسه` | بيانات الجلسة |
| `g` | `فلاسك.عام` | المخزن العام للطلب |
| `current_app` | `فلاسك.التطبيق_الحالي` | وكيل التطبيق |
| `jsonify` | `فلاسك.حول_json` | تحويل البيانات إلى JSON |
| `render_template` | `فلاسك.صيغ` | تصيير قالب Jinja2 |
| `redirect` | `فلاسك.حول` | إعادة التوجيه HTTP |
| `url_for` | `فلاسك.رابط_ل` | بناء رابط لنقطة نهاية |
| `abort` | `فلاسك.اجهض` | رمي استثناء HTTP |
| `flash` | `فلاسك.ومض` | رسائل فورية |
| `send_file` | `فلاسك.ارسل_ملف` | إرسال ملف |

### ديكوراتات المسارات (على كائن التطبيق)

| Python | العربية |
|--------|---------|
| `app.route` | `تطبيق.طريق` |
| `app.get` | `تطبيق.احصل` |
| `app.post` | `تطبيق.انشر` |
| `app.put` | `تطبيق.ضع` |
| `app.delete` | `تطبيق.احذف` |
| `app.patch` | `تطبيق.رقع` |
| `app.run` | `تطبيق.يعمل` |
| `app.errorhandler` | `تطبيق.معالج_الخطا` |
| `app.before_request` | `تطبيق.قبل_الطلب` |
| `app.after_request` | `تطبيق.بعد_الطلب` |

### مثال: تطبيق Flask كامل

```python
استورد فلاسك

تطبيق = فلاسك.فلاسك(__name__)

@تطبيق.طريق("/")
دالة الصفحه_الرئيسيه():
    ارجع فلاسك.صيغ("index.html", عنوان="مرحبا")

@تطبيق.طريق("/api/مستخدمون", methods=["GET"])
دالة قائمه_المستخدمين():
    مستخدمون = [
        {"id": 1, "اسم": "علي"},
        {"id": 2, "اسم": "سارة"},
    ]
    ارجع فلاسك.حول_json(مستخدمون)

@تطبيق.طريق("/api/مستخدمون", methods=["POST"])
دالة اضف_مستخدم():
    بيانات = فلاسك.طلب.json
    اذا ليس بيانات أو "اسم" ليس_في بيانات:
        فلاسك.اجهض(400)
    # حفظ في قاعدة البيانات...
    ارجع فلاسك.حول_json({"رسالة": "تم الإضافة"}), 201

@تطبيق.معالج_الخطا(404)
دالة صفحه_غير_موجوده(خطا):
    ارجع فلاسك.حول_json({"خطا": "الصفحة غير موجودة"}), 404

اذا __name__ == "__main__":
    تطبيق.يعمل(debug=صحيح)
```

## طلبات (`requests`)

### الاستيراد

```python
استورد طلبات
```

### أفعال HTTP

| Python | العربية | مثال |
|--------|---------|-------|
| `get` | `احصل` | `طلبات.احصل("https://...")` |
| `post` | `نشر` | `طلبات.نشر(url, json=...)` |
| `put` | `ضع` | `طلبات.ضع(url, data=...)` |
| `delete` | `احذف` | `طلبات.احذف(url)` |
| `patch` | `عدل` | `طلبات.عدل(url, json=...)` |
| `head` | `راس` | `طلبات.راس(url)` |
| `options` | `خيارات` | `طلبات.خيارات(url)` |
| `request` | `اطلب` | `طلبات.اطلب("GET", url)` |

### الفئات والإعدادات

| Python | العربية |
|--------|---------|
| `Session` | `جلسه` |
| `Response` | `استجابه` |
| `auth.HTTPBasicAuth` | `مصادقه_اساسيه` |
| `auth.HTTPDigestAuth` | `مصادقه_هضم` |
| `codes` | `رموز_الحاله` |
| `exceptions` | `استثناءات` |

### استثناءات requests

| Python | العربية |
|--------|---------|
| `RequestException` | `خطا_طلب` |
| `ConnectionError` | `خطا_اتصال` |
| `Timeout` | `خطا_مهله` |
| `HTTPError` | `خطا_بروتوكول` |
| `TooManyRedirects` | `خطا_كثير_تحويلات` |

### مثال كامل

```python
استورد طلبات

# طلب GET بسيط
استجابه = طلبات.احصل("https://api.github.com/users/github")
اطبع(استجابه.status_code)
بيانات = استجابه.json()
اطبع(f"المستخدم: {بيانات['name']}")

# إرسال بيانات JSON
رأس = {"Content-Type": "application/json", "Authorization": "Bearer TOKEN"}
استجابه2 = طلبات.نشر(
    "https://api.example.com/بيانات",
    json={"مفتاح": "قيمة"},
    headers=رأس,
    timeout=10
)
استجابه2.raise_for_status()

# جلسة مستمرة
مع طلبات.جلسه() كـ جلسه:
    جلسه.headers.update({"Authorization": "Bearer TOKEN"})
    ر1 = جلسه.احصل("https://api.example.com/نقطه1")
    ر2 = جلسه.احصل("https://api.example.com/نقطه2")
    اطبع(ر1.json(), ر2.json())
```

## أيو_هتب (`aiohttp`)

### الاستيراد

```python
استورد أيو_هتب
```

### المكونات الرئيسية

| Python | العربية | الوصف |
|--------|---------|--------|
| `ClientSession` | `جلسه_غير_متزامنه` | الجلسة غير المتزامنة |
| `ClientError` | `خطا_عميل` | استثناء عميل aiohttp |
| `ClientTimeout` | `مهله_عميل` | إعداد المهلة |
| `TCPConnector` | `اعدادات_اتصال` | إعداد اتصال TCP |
| `web.Application` | `تطبيق_ويب` | تطبيق الخادم |
| `web.run_app` | `تشغيل_خادم` | تشغيل الخادم |
| `web.Request` | `طلب_ويب` | كائن الطلب |
| `web.Response` | `استجابه_ويب` | كائن الاستجابة |
| `web.json_response` | `استجابه_جيسون` | استجابة JSON |
| `web.RouteTableDef` | `مسارات_ويب` | جدول المسارات |

### مثال: عميل HTTP غير متزامن

```python
استورد اتزامن
استورد أيو_هتب

غير_متزامن دالة جلب(url):
    مع أيو_هتب.جلسه_غير_متزامنه() كـ جلسه:
        انتظر جلسه.get(url) كـ استجابه:
            ارجع انتظر استجابه.json()

غير_متزامن دالة جلب_متوازي(روابط):
    مع أيو_هتب.جلسه_غير_متزامنه() كـ جلسه:
        مهام = [جلسه.get(رابط) لكل رابط في روابط]
        نتائج = انتظر اتزامن.اجمع(*مهام)
        ارجع [انتظر ن.json() لكل ن في نتائج]

# تشغيل
روابط = [
    "https://api.example.com/نقطه1",
    "https://api.example.com/نقطه2",
    "https://api.example.com/نقطه3"
]
نتائج = اتزامن.شغل(جلب_متوازي(روابط))
```

### مثال: خادم ويب aiohttp

```python
استورد اتزامن
استورد أيو_هتب

مسارات = أيو_هتب.مسارات_ويب()

@مسارات.get("/")
غير_متزامن دالة الرئيسيه(طلب):
    ارجع أيو_هتب.استجابه_ويب(text="مرحبا بك!")

@مسارات.get("/api/حاله")
غير_متزامن دالة الحاله(طلب):
    ارجع أيو_هتب.استجابه_جيسون({"حاله": "تعمل"})

تطبيق = أيو_هتب.تطبيق_ويب()
تطبيق.router.add_routes(مسارات)

اذا __name__ == "__main__":
    أيو_هتب.تشغيل_خادم(تطبيق, host="127.0.0.1", port=8080)
```

### الاستيراد

```python
استورد فاست_أبي
```

### المكونات الرئيسية

| Python | العربية | الوصف |
|--------|---------|--------|
| `FastAPI` | `تطبيق_سريع` | الصنف الرئيسي للتطبيق |
| `APIRouter` | `موجه_api` | تنظيم المسارات |
| `Query` | `استعلام` | معامل الاستعلام |
| `Path` | `مسار_معامل` | معامل المسار |
| `Body` | `جسم` | جسم الطلب |
| `Form` | `استماره` | بيانات النموذج |
| `Header` | `ترويسه` | ترويسة HTTP |
| `Cookie` | `كوكي` | ملف تعريف الارتباط |
| `File` | `ملف` | رفع ملف |
| `UploadFile` | `ملف_مرفوع` | ملف مرفوع |
| `Depends` | `يعتمد_علي` | حقن الاعتماديات |
| `HTTPException` | `خطا_http` | استثناء HTTP |
| `status` | `حالات_http` | رموز حالة HTTP |

### استجابات FastAPI

| Python | العربية |
|--------|---------|
| `JSONResponse` | `رد_json` |
| `HTMLResponse` | `رد_html` |
| `FileResponse` | `رد_ملف` |
| `StreamingResponse` | `رد_بث` |
| `RedirectResponse` | `رد_توجيه` |

### مثال: تطبيق FastAPI كامل

```python
استورد فاست_أبي
من pydantic استورد BaseModel

تطبيق = فاست_أبي.تطبيق_سريع(
    title="واجهة المستخدمين",
    description="API لإدارة المستخدمين",
    version="1.0.0"
)

# نموذج Pydantic
صنف مستخدم(BaseModel):
    id: عدد_صحيح | لا_شيء = لا_شيء
    اسم: نص
    بريد: نص
    نشط: منطقي = صحيح

# قاعدة بيانات وهمية
مستخدمون_db: قاموس[عدد_صحيح, مستخدم] = {}
عداد_id = 0

@تطبيق.احصل("/")
غير_متزامن دالة الجذر():
    ارجع {"رسالة": "مرحباً بك في واجهة المستخدمين"}

@تطبيق.احصل("/مستخدمون/")
غير_متزامن دالة قائمه_المستخدمين(
    تخطي: عدد_صحيح = فاست_أبي.استعلام(default=0, ge=0),
    حد: عدد_صحيح = فاست_أبي.استعلام(default=10, le=100)
):
    ارجع قائمه(مستخدمون_db.values())[تخطي:تخطي+حد]

@تطبيق.انشر("/مستخدمون/", response_model=مستخدم)
غير_متزامن دالة انشئ_مستخدم(مستخدم: مستخدم):
    عالمي عداد_id
    عداد_id += 1
    مستخدم.id = عداد_id
    مستخدمون_db[عداد_id] = مستخدم
    ارجع مستخدم

@تطبيق.احصل("/مستخدمون/{معرف}")
غير_متزامن دالة اجلب_مستخدم(معرف: عدد_صحيح):
    اذا معرف ليس_في مستخدمون_db:
        ارفع فاست_أبي.خطا_http(status_code=404, detail="المستخدم غير موجود")
    ارجع مستخدمون_db[معرف]
```

# Web Aliases (English summary)

| Arabic import | Python module | Purpose |
|---|---|---|
| `فلاسك` | `flask` | Lightweight web framework; WSGI |
| `طلبات` | `requests` | Synchronous HTTP client |
| `أيو_هتب` | `aiohttp` | Async HTTP client + server |
| `فاست_أبي` | `fastapi` | Async API framework with auto OpenAPI docs |

All four packages must be installed separately (`pip install flask requests aiohttp fastapi`). The alias runtime resolves Arabic import names to the real packages automatically.

See also: [stdlib-aliases.md](stdlib-aliases.md) for `اتزامن` (asyncio) which aiohttp and fastapi depend on.
