# مرجع الكلمات المفتاحية

هذه الصفحة تعرض الكلمات المفتاحية المعتمدة في القاموس النشط `ar-v2`. البدائل التاريخية مثل `كـ` و`هو` موثقة في المعجم فقط، ولا تستخدم في أمثلة `ar-v2` الجديدة.

## التحكم في التدفق

| العربية | Python | مثال |
|---|---|---|
| `إذا` | `if` | `إذا س > 0:` |
| `وإلا_إذا` | `elif` | `وإلا_إذا س == 0:` |
| `وإلا` | `else` | `وإلا:` |
| `طالما` | `while` | `طالما صحيح:` |
| `لكل` | `for` | `لكل عنصر في عناصر:` |
| `في` | `in` | `إذا اسم في أسماء:` |
| `يكون` | `is` | `إذا قيمة يكون لاشيء:` |
| `ليس` | `not` | `إذا ليس جاهز:` |
| `و` | `and` | `إذا أ و ب:` |
| `أو` | `or` | `إذا أ أو ب:` |
| `اكسر` | `break` | `اكسر` |
| `استمر` | `continue` | `استمر` |
| `مرر` | `pass` | `مرر` |
| `حاول` | `try` | `حاول:` |
| `استثناء` | `except` | `استثناء خطأ_قيمة:` |
| `أخيرا` | `finally` | `أخيرا:` |
| `ارفع` | `raise` | `ارفع خطأ_قيمة("رسالة")` |
| `مع` | `with` | `مع افتح("ملف.txt") باسم ف:` |
| `باسم` | `as` | `استورد بانداس باسم بد` |

## التعريف والاستيراد

| العربية | Python | مثال |
|---|---|---|
| `دالة` | `def` | `دالة احسب(س):` |
| `صنف` | `class` | `صنف عداد:` |
| `ارجع` | `return` | `ارجع نتيجة` |
| `سلم` | `yield` | `سلم قيمة` |
| `استورد` | `import` | `استورد رياضيات` |
| `من` | `from` | `من رياضيات استورد جذر` |
| `عام` | `global` | `عام عداد` |
| `غير_محلي` | `nonlocal` | `غير_محلي قيمة` |
| `لامدا` | `lambda` | `لامدا س: س * 2` |
| `أكد` | `assert` | `أكد س > 0` |
| `احذف` | `del` | `احذف عنصر` |

## غير متزامن ومطابقة

| العربية | Python | مثال |
|---|---|---|
| `غير_متزامن` | `async` | `غير_متزامن دالة جلب():` |
| `انتظر` | `await` | `نتيجة = انتظر طلب()` |
| `طابق` | `match` | `طابق أمر:` |
| `حالة` | `case` | `حالة "خروج":` |

## القيم الحرفية

| العربية | Python |
|---|---|
| `صحيح` | `True` |
| `خطأ` | `False` |
| `لاشيء` | `None` |

## مثال كامل

```python
حاول:
    نتيجة = 10 / 0
استثناء خطأ_قسمة_صفر:
    اطبع("لا يمكن القسمة على صفر")
أخيرا:
    اطبع("انتهت العملية")
```
