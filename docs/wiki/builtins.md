# الدوال المدمجة — Built-ins Reference

هذه القائمة الكاملة للدوال والأنواع المدمجة في لغة الثعبان مع مكافئاتها في Python القياسي. كل اسم عربي يمر تلقائياً عبر مُطبّع المعرّفات قبل البحث (انظر [قواعد التطبيع](normalization.md)).

| Python | العربية | مثال |

| `bool` | `منطقي` | `اذا منطقي(1):` |
| `list` | `قائمه` | `ا = قائمه(مدى(5))` |
| `dict` | `قاموس` | `د = قاموس(ا=1, ب=2)` |
| `tuple` | `صف` | `ث = صف([1, 2, 3])` |
| `set` | `مجموعه` | `م = مجموعه([1, 1, 2])` |
| `frozenset` | `مجموعه_ثابته` | `ف = مجموعه_ثابته([1, 2])` |
| `bytes` | `بايتات` | `ب = بايتات(4)` |
| `bytearray` | `مصفوفه_بايتات` | `م = مصفوفه_بايتات(b"hello")` |
| `complex` | `عدد_مركب` | `ع = عدد_مركب(2, 3)` |

## الدوال المدمجة

### الإدخال والإخراج

### التحويل بين الأنواع

| Python | العربية | مثال |
|--------|---------|-------|
| `int` | `عدد_صحيح` | `عدد_صحيح("42")` → `42` |

## أمثلة متكاملة

### معالجة قائمة الأرقام

```python
ارقام = [15, -3, 42, 0, -7, 8]

# الحد الأقصى والأدنى
اطبع("الأكبر:", الأكبر(ارقام))
اطبع("الأصغر:", الأصغر(ارقام))

# المجموع والمطلق
مجموع_المطلقات = مجموع(طبق(مطلق, ارقام))
اطبع("مجموع القيم المطلقة:", مجموع_المطلقات)

# الأرقام الموجبة فقط
موجبه = قائمه(فلتر(لامدا س: س > 0, ارقام))
اطبع("الأرقام الموجبة:", موجبه)

# مرتبة تنازلياً
مرتبه = مرتب(ارقام, reverse=صحيح)
اطبع("مرتبة:", مرتبه)
```

### التحقق من الأنواع

```python
دالة وصف_كائن(ك):
    اطبع(f"النوع: {نوع(ك).__name__}")
    اطبع(f"المعرف: {معرف(ك)}")
    اطبع(f"التمثيل: {تمثيل(ك)}")
    
    اذا من_نوع(ك, (قائمه, صف, مجموعه)):
        اطبع(f"الطول: {طول(ك)}")

وصف_كائن([1, 2, 3])
وصف_كائن("مرحبا")
وصف_كائن(42)
```

### استخدام enumerate و zip

```python
اسماء = ["علي", "سارة", "محمد"]
درجات = [85, 92, 78]

لكل رقم, (اسم, درجه) في رقم(اضغط(اسماء, درجات), start=1):
    اطبع(f"{رقم}. {اسم}: {درجه}/100")
```
