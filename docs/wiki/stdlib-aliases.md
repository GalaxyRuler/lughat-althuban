# Stdlib Aliases (English summary)

31 standard-library modules are available under Arabic names. Canonical import names and compatibility aliases are authored in `lexicon/libraries.toml` and generated into runtime TOMLs.

Use the generated Arabic aliases index for exhaustive entries. Prefer canonical names in new docs and examples; compatibility names exist only to keep older snippets working.
