# مكتبات الوسائط والألعاب — Media & Game Aliases

لغة الثعبان توفر أسماء عربية لمكتبات الوسائط والألعاب التي يمكن استخدامها في
برامج `.apy` محلية أو في أمثلة الملعب المرئية.

---

## جدول المكتبات

| الاسم العربي | Python | الوصف |
|-------------|--------|--------|
| `بيلو` | `PIL.Image` | فتح الصور وتعديلها وحفظها عبر Pillow |
| `لعبه`، `لعبة` | `pygame` | نوافذ، أسطح، رسم ثنائي الأبعاد، أحداث، وتصادم عبر pygame-ce |

> الأسماء المعتمدة تأتي من `lexicon/libraries.toml`، وتظهر الخريطة المولدة في
> `arabicpython/aliases/pillow.toml` و`arabicpython/aliases/pygame.toml`.

---

## Pygame بالعربية

### الاستيراد

```python
استورد لعبه
```

### رسم نافذة بسيطة

```python
استورد لعبه

لعبه.ابدا()
نافذه = لعبه.نافذه((٣٢٠، ١٨٠))
لعبه.عنوان_النافذه("نافذة عربية")

مربع = لعبه.مستطيل(٤٠، ٧٠، ٣٢، ٣٢)
لعبه.املا(نافذه، (٥، ١١، ٢٠))
لعبه.ارسم_مستطيل(نافذه، (٩٤، ٢٣٤، ٢١٢)، مربع)
لعبه.اعرض()

ساعه = لعبه.ساعه()
لعبه.انتظر_اطار(ساعه، ٦٠)
لعبه.اغلق()
```
