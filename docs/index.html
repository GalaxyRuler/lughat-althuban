لغة الثعبان

بايثون بالعربية

لهجة عربية أولى فوق CPython تتيح كتابة ملفات .apy بكلمات مفتاحية ودوال مدمجة واستثناءات وأسماء مكتبات عربية، مع معجم موحد ومترجم عكسي وأثر أخطاء عربي.

افتح الملعب العب قفزة المدينة شغّل برنامجا كبيرا جرّب لعبة عربية تصفح الأمثلة المستودع على GitHub

بداية سريعة

python -m pip install lughat-althuban
ثعبان -c "اطبع('مرحبا بالعالم')"

الوثائق

دليل البدء

خطوات التثبيت وتشغيل أول برنامج عربي.

المعجم العربي الموحد

الكلمات المفتاحية والدوال والاستثناءات والمكتبات، بما فيها المكتبة القياسية والذكاء الاصطناعي، في مرجع مولد واحد.

معرض الأمثلة

أمثلة تعمل في الملعب وروابط لأمثلة المكتبة القياسية والذكاء الاصطناعي والتشغيل المحلي.

سياسات اللغة

قواعد تسمية المصطلحات والأسماء المستعارة والاستثناءات.

التطبيع والسلامة النصية

قواعد الهمزات والتشكيل واتجاه النص والمحارف الخفية.

استكشاف المشكلات

حلول سريعة لمشكلات التثبيت والترميز وتشغيل الأدوات.

ويكي المشروع

مراجع تفصيلية للأدوات والبنية والأسئلة الشائعة.

الأدوات

  • ثعبان لتشغيل ملفات .apy والسطر المباشر.
  • ثعبان ترجمة-عكسية لتحويل بايثون إلى لغة الثعبان.
  • --tracebacks=arabic|english|mixed لاختيار وضع أثر الأخطاء.
  • ثعبان نسّق لتنسيق الشفرة العربية.
  • ثعبان راجع لفحص الأسلوب والتنبيهات.
  • نواة Jupyter وإضافة VS Code لتجربة تحرير عربية أفضل.
